Abstract

We present CogScene, a novel framework for holistic 3D scene understanding that draws inspiration from human cognitive processes. Unlike existing methods that treat geometric reconstruction and semantic understanding as independent tasks, CogScene integrates top-down cognitive priors with bottom-up visual features in a unified architecture. Our method processes monocular video and produces temporally consistent 3D semantic scene graphs that capture object identities, spatial relations, and functional affordances over time.

A key insight is that top-down scene expectations — learned from large-scale commonsense knowledge — substantially improve the accuracy and robustness of bottom-up visual parsing under occlusion, motion blur, and illumination changes. CogScene introduces a Cognitive Prior Module (CPM) that iteratively updates scene graph beliefs using a cross-attention mechanism between visual tokens and structured scene hypotheses. Extensive experiments on ScanNet, Replica, and our newly introduced CogBench dataset demonstrate state-of-the-art performance across object detection, scene graph generation, and tracking metrics, while requiring 60% less computation than competing approaches.

CogScene architecture overview. Given monocular video, a video backbone extracts spatiotemporal features. The Cognitive Prior Module (CPM) (highlighted) iteratively refines scene graph beliefs via cross-attention between visual tokens and structured scene hypotheses drawn from a commonsense knowledge base. A lightweight decoder produces 3D bounding boxes, semantic labels, and pairwise relation logits per frame, enforced to be temporally consistent.
